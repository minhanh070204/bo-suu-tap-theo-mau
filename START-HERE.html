💡 Tips for Testing

  • Max width is 480px (iPhone-sized) - centered on desktop

  • Try the bottom navigation to switch between sections
  • Complete tasks to see point rewards
  • Use the back arrow to return to previous screens
  • Look for interactive elements (buttons, cards, icons)
  • Toast notifications appear for feedback
